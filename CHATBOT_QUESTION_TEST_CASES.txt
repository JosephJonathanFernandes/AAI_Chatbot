================================================================================
              CHATBOT QUESTION TYPES & EDGE CASES TEST SUITE
              Comprehensive Query Testing Guide
================================================================================

This document outlines comprehensive test cases for all types of questions,
edge cases, and scenario variations that the chatbot should handle.

Format:
- Question ID: Unique identifier
- Question Type: Category of question
- Test Query: Exact query to send
- Expected Intent: What intent should be detected
- Expected Behavior: How chatbot should respond
- Edge Case Notes: Special considerations
