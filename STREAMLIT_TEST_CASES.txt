================================================================================
                    STREAMLIT CHATBOT TEST CASES
                    Comprehensive Testing Guide
================================================================================

This document outlines all test cases to verify the working of the chatbot on Streamlit.
Each test includes:
- Test ID: Unique identifier
- Test Name: Description of what is being tested
- Steps: Action sequence to execute the test
- Expected Result: What should happen
- Acceptance Criteria: How to verify success
